Project: Sales Performance Analysis
Objective:
To analyze monthly sales data to identify trends,
distribution patterns, and outliers that impact business performance.

Dataset:
Monthly sales data(CSV format) containing sales values by month

Tools & Libraries:
- Python
- Pandas
- Matplotlib
- Seaborn

Key Analytics Performed:
- Data cleaning and validation
- Descriptive statistics
- Trend analysis
- Sales comparison across months
- Business insights generation

Project Structure:
- 'sales_analysis.py' -> Data analysis logic
- 'visuals/' -> Visualization scripts
- 'outputs/' -> Business insights & reports
- 'Cleaned_sales_data.csv' -> Final cleaned dataset
    
Status:
Completed (Exploratory Data Analysis)
